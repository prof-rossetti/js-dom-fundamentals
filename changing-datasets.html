Changing Datasets

Objectives

Satisfy the following objectives using vanilla JavaScript:

  1. Add an a element, nested inside an h3 element, nested inside the div#my-container element.
  2. Configure the text of the a element to display the name of the course (myCourse).
  3. Configure the a element's href attribute to hyperlink to the course link (myCourse).
  4. Configure the `select#course-selector` element such that when it's value is changed, the proper course is displayed (myCourse vs otherCourse).
